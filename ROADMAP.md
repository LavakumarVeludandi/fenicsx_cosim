# ROADMAP

This roadmap lists user-facing priorities for `fenicsx-cosim`.

## Near-term priorities

- Improve interpolation options for non-conforming interfaces:
  - Radial Basis Function (RBF) mapping
  - Linear mapping option in addition to nearest-neighbor
- Add strong (implicit) coupling support with fixed-point sub-iterations
- Add adaptive coupling acceleration (Aitken relaxation)
- Expand test coverage with mapping-accuracy and convergence-focused tests

## Ecosystem and usability

- Multi-solver workflows beyond FEniCSx/FEniCSx pairings
- Better diagnostics and convergence reporting in coupling loops
- More examples for thermo-mechanical and FE² pipelines
- Additional benchmark and validation cases for interface conservation

## Documentation and community

- Publish comparative guidance for coupling choices and use cases
- Expand tutorial notebooks and reproducible benchmark scripts
- Improve contribution pathways and issue triage templates
